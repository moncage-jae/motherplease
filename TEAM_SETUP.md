# HeavenOrHell — 팀원 환경 세팅 가이드

---

## 1. Git LFS 설치

### 1-1. 다운로드 및 설치
1. https://git-lfs.com 접속 후 다운로드
2. 설치 파일 실행 (Next 계속 눌러서 완료)

### 1-2. LFS 활성화
1. `Windows + R` → `cmd` 입력 → 확인
2. 아래 명령어 입력 후 엔터

```
git lfs install
```

---

## 2. 소스트리 세팅

### 2-1. 소스트리 로그인
1. https://sourcetreeapp.com 접속 후 다운로드 및 설치
2. 실행 후 **계정 추가** 클릭
3. GitHub 아이디로 로그인

---

### 2-2. SSH 키 생성
1. `Windows + R` → `cmd` 입력 → 확인
2. 아래 명령어 입력 후 엔터

```
ssh-keygen -t rsa -b 4096 -C "GitHub 로그인 이메일"
```

3. 경로 묻는 메시지 → 그냥 **엔터** (기본 경로에 저장)
4. 비밀번호 묻는 메시지 → **엔터 두 번** (비밀번호 없이 생성)
5. 아래 경로에 파일 두 개 생성됐는지 확인

```
C:\Users\[사용자이름]\.ssh\
    id_rsa        ← 개인 키 (외부 공유 금지)
    id_rsa.pub    ← 공개 키
```

---

### 2-3. PPK 파일로 변환

> 소스트리는 `.ppk` 형식의 SSH 키를 사용하기 때문에 변환이 필요합니다.

1. **PuTTYgen** 프로그램 다운로드 및 실행
   - https://www.putty.org 에서 다운로드
2. **Load** 버튼 클릭
3. 파일 형식을 **모든 파일 (`*.*`)** 로 변경 후 `id_rsa` 파일 선택
4. **Save private key** 클릭 → `.ssh` 폴더 안에 `.ppk` 파일로 저장

---

### 2-4. GitHub에 SSH 공개 키 등록
1. https://github.com 로그인
2. 우상단 프로필 사진 클릭 → **Settings**
3. 좌측 메뉴 **SSH and GPG keys** 클릭
4. **New SSH key** 클릭
5. `id_rsa.pub` 파일을 메모장으로 열어서 내용 전체 복사 후 붙여넣기
6. **Add SSH key** 클릭

---

### 2-5. 소스트리에 PPK 키 등록
1. 소스트리 상단 메뉴 **도구** → **SSH 키 관리자** 실행
2. 앞서 저장한 `.ppk` 파일 선택
