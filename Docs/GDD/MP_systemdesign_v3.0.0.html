Chapter 02

조작 시스템

플레이어가 게임과 상호작용하는 유일한 수단을 정의합니다. 모든 입력은 왼손 하나의 움직임으로만 처리되며, 이 제약 자체가 게임의 핵심 공포 메커닉입니다.

2.1 설계 원칙 — 왼손 단일 조작
핵심 원칙

플레이어는 사지마비 환자입니다. 왼손을 제외한 모든 신체 부위는 조작 불가입니다. 이 조작 제약은 UX 한계가 아닌 의도된 무력감 연출입니다. 어떠한 이유로도 이동 속도를 "편의상" 높이거나 추가 입력 수단을 허용해서는 안 됩니다.

조작 개요
입력 수단
방향키(이동) + 마우스 휠(상호작용) — 2종 고정
조작 대상
왼팔 전체 (상완 → 전완 → 손목 → 손가락 순으로 물리 연산)
조작 속도
모든 입력에 극한의 지연 적용 — 체감 기준 10초에 1cm 수준
속도 변화
게임 전 구간 동일 — 이벤트·긴장 상황에서도 속도 보정 없음
레퍼런스
The Longing (2020) — 실시간 경과 기반 극저속 조작 방식
2.2 방향키 입력 — 팔 이동

방향키 4방향 입력으로 왼팔 전체를 전후좌우로 이동시킵니다. 입력은 연속 누름(Hold) 방식으로만 동작하며, 탭 입력은 무시합니다.

4.1.2 — Flag 초기화 규칙

트리거 Flag를 읽는 시스템은 현재 미니게임 시스템(4장)과 독백 시스템(7장) 두 곳입니다. 연출(8장)·오디오(10장)는 Flag를 사용하지 않습니다. 챕터 재진입 시 두 시스템의 Flag를 함께 초기화하여 독백과 미니게임 모두 처음부터 재생되도록 보장합니다.

초기화 누락 시 발생하는 문제

Flag가 초기화되지 않으면 불러오기(챕터 재플레이)로 챕터 재진입 시 Flag가 이미 true 상태로 남아 있어 미니게임과 독백이 기동되지 않습니다.
