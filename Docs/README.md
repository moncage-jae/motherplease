# Docs/

기획 및 기술 문서 전체를 관리하는 폴더입니다.
기획자가 주로 관리합니다.

## 구조

| 폴더 | 내용 |
|------|------|
| `GDD/` | 게임 디자인 문서, 기획서, 시스템 설계 |
| `TDD/` | 기술 설계 문서, 아키텍처, API 문서 |
| `References/` | 레퍼런스 이미지, 무드보드, 링크 모음 |
| `MeetingNotes/` | 회의록 |

## 파일 명명 규칙

```
회의록_20260327.md
시스템기획서_v1.1.html
```

## 규칙

- 문서는 `.md` 또는 `.html` 형식을 사용합니다
- 버전 또는 날짜를 파일명에 포함합니다
- 이 폴더는 일반 Git으로 관리됩니다 (LFS 제외)
