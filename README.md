# Mother. Please — 공유 저장소

## 폴더 구조

| 폴더 | 용도 | 주 담당 |
|------|------|---------|
| `RawAssets/` | 원본 소스 파일 (.blend, .psd, .fbx 등) | 3D모델러, 기획, VFX |
| `Docs/` | 기획서, 기술문서, 레퍼런스, 회의록 | 기획 (전원 열람) |
| `ExternalShare/` | 외부 공유용 파일 (프레스킷, 외주) | 기획 |
| `_WIP/` | 개인 임시 작업 폴더 | 각자 본인 폴더만 사용 |

## 규칙

- **에셋은 용도/유형별로 분류합니다.** 사람 이름이 아닌 에셋 종류 기준으로 폴더에 넣어주세요.
- **파일명 컨벤션**: `에셋유형_이름_버전.확장자` (예: `CH_Patient_v02.blend`)
- **대용량 바이너리 파일**(.blend, .psd, .fbx, .wav 등)은 Git LFS로 관리합니다.
- **완성된 파일만 해당 폴더에 넣고**, 작업 중인 파일은 `_WIP/본인폴더/`에 둡니다.
- **커밋 메시지**는 한글로 간결하게 작성합니다. (예: `환자 캐릭터 모델 v2 추가`)
