# RawAssets/

엔진에 임포트하기 전 **원본 작업 파일** 보관 폴더입니다.
Git LFS로 관리됩니다.

## 구조

| 폴더 | 내용 |
|------|------|
| `3D/` | 3D 원본 파일 (.blend, .fbx, .ma 등) |
| `2D/` | 2D 원본 파일 (.psd, .ai 등) |
| `VFX/` | VFX 소스 파일 |
| `Audio/` | 원본 사운드 파일 (.wav, DAW 프로젝트 등) |

## 파일 명명 규칙

```
에셋유형_이름_버전.확장자

예시:
  CH_Patient_v02.blend       ← 캐릭터
  TX_WallBrick_Diffuse.psd   ← 텍스처
  AU_Ambient_Ward_v01.wav    ← 오디오
```

## 규칙

- 작업이 완료된 파일만 이곳으로 옮깁니다
- 작업 중인 파일은 `_WIP/본인폴더/` 에 보관
- 이름 기준이 아닌 **에셋 종류 기준**으로 분류합니다
