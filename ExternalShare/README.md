# ExternalShare

팀 외부에 공유할 수 있는 파일을 보관합니다.

## 하위 폴더

| 폴더 | 내용 |
|------|------|
| `PressKit/` | 프레스킷 — 외신, 유튜버, 미디어 접근 시 공유할 파일 |
| `Outsource/` | 외주 작업 시 공유할 파일 |

## 규칙

- **팀원이 아닌 사람에게 공유하기 애매한 파일은 넣지 마세요.** (빌드 파일, 내부 문서 등)
- 프레스킷에는 로고, 스크린샷, 트레일러, 팩트시트 등을 포함합니다.
- 외주 파일에는 작업 가이드, 스타일 가이드, 필요한 레퍼런스를 넣습니다.
